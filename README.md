Prototype for an interactive study of tempo inflections in Grieg's 1903 recording
of 'Sommerfugl'.

Requirements
------------
- [Node.js](nodejs.org)

Installation
------------

1. `git clone https://github.com/pfefferniels/grieg-inflections`
2. `npm install`
3. `npm start` and open `localhost:3000` in the browser.
